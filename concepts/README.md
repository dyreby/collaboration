# Concepts

Mental models for how I think and work.
They're here so collaborators and agents understand the reasoning behind my choices.

All models are wrong, some are useful. These are the ones I find useful today.

[`principles.md`](principles.md) is foundational. Values I hold across everything.

Everything else is scoped to a topic. Files are named `{topic}.md` or `{topic}-{aspect}.md`, like `git-practices.md` or `rust-preferences.md`.
