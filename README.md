What someone means and what they say are never quite the same.
What someone says and what the other person hears are never quite the same either.
I don't think this gap ever fully closes, but it can be narrowed.

This is a collection of models I've found useful for that work.
They are shaped in part by what I've learned collaborating on software with AI agents, but are not limited to it.

## What's here

### [Philosophy](philosophy/)

Philosophical models about collaboration.
Why the gap exists, what that means for alignment between collaborators, and how agents navigate lossy compression of the world around them.

### [Concepts](concepts/)

Shared mental models for working with my agents.
Some are concrete enough to use directly; others are better synthesized into task-specific guidance.
Either way, they exist so I don't have to re-explain the same idea every session.

### [Roles](roles/)

Agent personas for focused tasks like planning work, writing code, and reviewing changes.
